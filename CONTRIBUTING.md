# Contributing

Mvm is a free and open-source project, and your feedback and contributions are
needed and always welcome.

[Issues](https://github.com/mvm-sh/mvm/issues) and [pull requests](https://github.com/mvm-sh/mvm/pulls)
are opened at https://github.com/mvm-sh/mvm.


## Documentation

- [docs/index.md](docs/index.md) — documentation entry point
- [docs/architecture.md](docs/architecture.md) — pipeline, memory model, key design decisions
- [docs/modules/](docs/modules/) — per-package reference
- [docs/decisions/](docs/decisions/) — architecture decision records (ADRs)
